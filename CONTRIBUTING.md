# Contributing to Focus Tools

Thanks for your interest in contributing!

## Reporting Bugs

Open an [issue](../../issues) with:
- A clear description of the problem
- Steps to reproduce it
- Your browser name and version

## Suggesting Features

Open an [issue](../../issues) describing the feature and why it would be useful.

## Submitting Code

1. Fork the repository
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Test the extension locally — load it as an unpacked extension and verify your changes work
5. Commit your changes with a clear message
6. Push to your fork and open a Pull Request

Please describe what your PR does and why in the PR description.
